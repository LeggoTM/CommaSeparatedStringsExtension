# CommaSeparatedStringsExtension
An extension to convert a list to comma separated strings enclosed in single quotes.

## Installation
- Download the latest `CommaSeparatedStringsExtension.zip` file from the [Github release page](https://github.com/LeggoTM/CommaSeparatedStringsExtension/releases).
- Unzip the `CommaSeparatedStringsExtension.zip` file.
- In Chrome browser, go to `chrome://extensions` page and switch on developer mode. This enables the ability to locally install a Chrome extension.
<img src="https://github.com/LeggoTM/CommaSeparatedStringsExtension/blob/main/readme_assets/load_unpacked.png" width="60%" height="60%"/>

- Now click on the `LOAD UNPACKED` and browse to the `unzipped folder`.
- This will install the extension to your browser.
